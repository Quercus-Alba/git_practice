NOTE 2: An em is equal to the current font-size, for example, if the font-size of the document is 12pt, 1em is equal to 12pt. Ems are scalable, so 2em (2 x 12pt) would equal 24pt, 0.5em (0.5 x 12pt) would equal 6pt.
